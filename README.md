## 📌 Project Overview

This project is an end-to-end Healthcare Sales Analytics solution built using **Google BigQuery, SQL, and Power BI**.

The objective is to analyze more than **250,000 pharmaceutical sales records** to uncover business insights regarding:

- Sales Performance
- Customer Behavior
- Product Performance
- Geographic Sales Trends
- Sales Team Performance

Future phases of the project include **Python-based Exploratory Data Analysis (EDA), Machine Learning, and a Streamlit Web Application**.
